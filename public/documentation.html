API Documentation

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON object holding data about all movies.
Return data about a single movie by title to the user. /movies/[Title] GET None A JSON object holding data about a single movie, containing description, genre, director, image URL, whether it’s featured. 
{
                "id": 1,
                "name": "Lord of the Rings",
                "description": "Is an epic fantasy adventure...",
                "genre": "Fantasy",
                "director": "Peter Jackson",
                "imageUrl": "https://example.com/lord_of_the_rings.jpg",
                "isFeatured": true
              }
Return data about a genre (description) by name/title (e.g., “Thriller”). /genres/[Name] GET None A JSON object holding a description of a selected genre. 
{
                "genre": "Action",
                "description": "Action films are characterized by..."
              }
Return data about a director (bio, birth year, death year) by name. /directos/[Name] GET None A JSON object holding data about a film director, containing bio, birth year, death year. Example: 
{
            "name": "Steven Spielberg",
            "bio": "Steven Spielberg is an American film director, producer, and screenwriter. Films: Jaws,
              E.T. the Extra-Terrestrial, Jurassic Park,Schindler's List, and Saving Private Ryan.",
            "birthYear": 1946,
            "deathYear": null
            }
Allow new users to register. /users POST A JSON object holding data about the user credentials(username, password), structured like: 
{
  "username": "example_user",
  "password": ""example_password"
}
 A JSON object holding data about the user that has been created, including an ID: 
{
                "userId":"123-123-12334",
                "username": "example_user",
                "password": "example_password"
              }
Allow users to update their user info (username). /users/[userId] PUT A JSON object holding a new username. 
              {
                "username": "new_username"
              }
A text message indicating the username has been updated.
Allow users to add a movie to their list of favorites. /users/[userId]/favorites POST A JSON object holding a movie data. Example: 
{
              "movieId": 456,
              "title": "The Shawshank Redemption",
              "genre": "Drama",
              "director": "Frank Darabont",
              "imageUrl": "https://example.com/shawshank_redemption_poster.jpg"
            }
 A text message indicating that the movie has been added to favourites.
Allow users to remove a movie from their list of favorites. users/[userId]/favorites/[movieId] DELETE None A text message indicating that the movie has been removed from favourites.
Allow existing users to deregister. users/[userId] DELETE None A text message indicating that the user has been removed.
